# QLineSec

Bu projenin amacı, Linux sistemlerinde çalışan systemd servislerini anlık olarak izlemek, bu servislerin durumlarını ve log kayıtlarını analiz ederek olası sistem problemlerini erken aşamada tespit etmektir.

Araç, özellikle sunucu yönetimi, DevOps ve siber güvenlik süreçlerinde sistem sürekliliğini sağlamak için geliştirilmiştir.
